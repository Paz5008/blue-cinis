// turbo-all

1. Vérification des types TypeScript
```bash
npm run typecheck
```

2. Vérification du linting
```bash
npm run lint
```

3. Build de production
```bash
npm run build
```

4. Si des erreurs sont détectées, les corriger avant de poursuivre.
